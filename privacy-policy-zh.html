爱码视 (AI Code View) 隐私政策

最后更新：2024年1月1日

1. 信息收集

爱码视 (AI Code View) 浏览器扩展致力于保护您的隐私。我们收集的信息类型包括：

  • 浏览器基本信息：浏览器版本、操作系统类型
  • 使用数据：扩展功能使用频率、崩溃报告
  • 代码显示设置：您在扩展中保存的主题偏好、字体大小设置

重要说明：我们不会收集、存储或传输您的代码内容、个人信息或敏感数据。

2. 信息使用

我们使用收集的信息用于：

  • 改进扩展的功能和用户体验
  • 修复错误和技术问题
  • 分析使用趋势以优化功能
  • 提供客户支持服务

2. 数据存储

所有数据都存储在您的本地浏览器中：

  • 设置和偏好保存在浏览器的本地存储中
  • 不会将数据发送到外部服务器
  • 您可以随时清除浏览器数据来删除这些信息

3. 第三方服务

爱码视 (AI Code View) 可能使用以下第三方服务：

  • Google Analytics：用于收集匿名的使用统计数据，帮助我们了解扩展的使用情况并改进产品
  • Google 收集的信息：
    • 浏览器类型和版本
    • 操作系统信息
    • 扩展功能使用频率
    • 匿名用户标识符

重要说明：目前本扩展未提供内置的用户控制选项来禁用 Google Analytics。Google Analytics 的使用受其隐私政策约束。您可以通过浏览器的隐私设置或第三方广告拦截插件来限制数据收集。

4. 数据安全

我们采取以下措施保护您的数据：

  • 所有数据本地存储，无需网络传输
  • 不收集敏感个人信息
  • 定期更新扩展以修复安全漏洞

5. Cookie 使用

爱码视 (AI Code View) 不使用 Cookie 或任何类似的跟踪技术。

6. 儿童隐私保护

我们不会有意收集 13 岁以下儿童的个人信息。如果您是家长或监护人，并认为您的孩子向我们提供了信息，请联系我们。

7. 政策更新

我们可能会不时更新此隐私政策。重大变更时，我们会在扩展内或通过其他方式通知您。建议您定期查看此政策以了解最新信息。

8. 联系我们

如果您对此隐私政策有任何疑问或建议，请通过以下方式联系我们：

GitHub：http://github.com/millylee/ai-code-view

9. 用户权利

您拥有以下权利：

  • 查看浏览器中存储的扩展数据
  • 清除浏览器中的扩展数据
  • 卸载扩展以停止所有数据收集
  • 选择不参与匿名的使用数据收集
