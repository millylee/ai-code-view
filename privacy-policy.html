AI Code View (爱码视) Privacy Policy

Last Updated: January 1, 2024

1. Information Collection

AI Code View (爱码视) browser extension is committed to protecting your privacy. The types of information we collect include:

  • Basic Browser Information: Browser version, operating system type
  • Usage Data: Extension feature usage frequency, crash reports
  • Code Display Settings: Theme preferences, font size settings saved in the extension

Important Note: We do not collect, store, or transmit your code content, personal information, or sensitive data.

2. Information Use

We use the collected information for:

  • Improving extension functionality and user experience
  • Fixing bugs and technical issues
  • Analyzing usage trends to optimize features
  • Providing customer support services

3. Data Storage

All data is stored locally in your browser:

  • Settings and preferences are saved in browser's local storage
  • No data is sent to external servers
  • You can clear browser data at any time to delete this information

4. Third-Party Services

AI Code View (爱码视) may use the following third-party services:

  • Google Analytics: Used to collect anonymous usage statistics to help us understand extension usage and improve the product
  • Information collected by Google:
    • Browser type and version
    • Operating system information
    • Extension feature usage frequency
    • Anonymous user identifiers

Important Notice: Currently, this extension does not provide built-in user control options to disable Google Analytics. Google Analytics usage is governed by their Privacy Policy. You may limit data collection through your browser's privacy settings or third-party ad-blocking extensions.

5. Data Security

We take the following measures to protect your data:

  • All data is stored locally with no network transmission
  • No sensitive personal information is collected
  • Regular extension updates to fix security vulnerabilities

6. Cookie Usage

AI Code View (爱码视) does not use cookies or any similar tracking technologies.

7. Children's Privacy Protection

We do not knowingly collect personal information from children under 13 years of age. If you are a parent or guardian and believe your child has provided us with information, please contact us.

8. Policy Updates

We may update this privacy policy from time to time. For significant changes, we will notify you within the extension or through other means. We recommend reviewing this policy periodically for the latest information.

9. Contact Us

If you have any questions or suggestions about this privacy policy, please contact us at:

GitHub: http://github.com/millylee/ai-code-view

10. User Rights

You have the following rights:

  • View extension data stored in your browser
  • Clear extension data from your browser
  • Uninstall the extension to stop all data collection
  • Opt out of anonymous usage data collection
